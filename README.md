## Overview
Serial(UART) Monitor for M5Stack.

Support for physical keyboards.
- M5Stack Faces Keyboard
- CardKB Keyboard

This project uses kazkojima's work. (https://github.com/kazkojima/TFT_vt100)

## Connection
     M5Stack     Devices(ex. MicroPython)
       GND         GND
       R0    ---   TxD
       T0    ---   RxD
